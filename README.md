## 🔥 가계북 개발 배경
현재에 가계부 서비스는 사용하기가 어렵다는 단점이 있습니다. 이에 저희 팀은 OCR 기술을 도입하여 사용자가 영수증만으로 손쉽게 소비내역을 등록하고 소비 내역을 바탕으로 카드를 추천 받을 수 있는 서비스를 개발하기 위해 이번 프로젝트를 기획하게 되었습니다.
